TypeScript SDK (generated)

How to generate
- Ensure the API is running and reachable, or export the spec to a file.
- Export OpenAPI spec to a file:
  - make openapi-export API_BASE=http://localhost:8000
- Generate the SDK using Dockerized openapi-generator:
  - make sdk-ts OPENAPI_SPEC=./openapi.json OUT_DIR=./sdk/ts

After generation
- Inspect and optionally adjust the configuration for injecting Bearer tokens:
  - For typescript-fetch generator, you can set a custom fetch or middleware to add the Authorization header.
- Build/publish as needed (package.json is generated by openapi-generator when npmName/npmVersion are provided).

Auth injection example (typescript-fetch)
- In the generated runtime.ts, you can set a default configuration with headers:
  import { Configuration } from './runtime';
  const cfg = new Configuration({ headers: { Authorization: `Bearer ${token}` } });
  // Pass cfg to API classes, e.g., new BookmarksApi(cfg)
